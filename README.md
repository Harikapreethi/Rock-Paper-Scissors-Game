# ✊✋✌️ Rock Paper Scissors (Python CLI Game)

A simple command-line implementation of the classic **Rock, Paper, Scissors** game using Python.
The user plays against the computer, which makes random choices.

---

## 🎮 How the Game Works

* The user selects one of:

  * Rock
  * Paper
  * Scissors
* The computer randomly selects a choice
* The winner is determined based on standard game rules:

  * Rock beats Scissors
  * Scissors beats Paper
  * Paper beats Rock

---

## 🚀 Features

* 🎯 User vs Computer gameplay
* 🔁 Continuous play option
* 🤖 Random computer choices
* ⚖️ Win / Lose / Tie logic
* ⚠️ Input validation for user choices
